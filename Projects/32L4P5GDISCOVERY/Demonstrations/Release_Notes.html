Purpose

This directory contains the demonstrations to illustrate the capabilities of the 32L4P5GDISCOVERY Kit.

Update History

Main Changes

  • Update demonstration using USB Device to support USB Device library v2.6.1
    • Update USBD_malloc and USBD_free macros definitions in usbd_conf.h file
    • Update data size parameter used by USBD_LL_Transmit() and USBD_LL_PrepareReceive() functions to uint32 in usbd_conf.c file
    • Update USBD_MAX_STR_DESC_SIZ and MSC_MEDIA_PACKET definitions to enable link step with IAR and Keil IDE
    • Reduce heap allocated in RAM to Jpeg library from 304Kb to 300Kb in order to enable link step with STM32CubeIDE
  • Update demonstration to support the latest version of HAL OSPI driver for L4+ devices
    • Remove WRAP configuration
    • Add missing DLYBYP and MAXTRAN configurations
